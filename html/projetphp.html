Projet PHP

Ce projet php a pour but de mettre en place une base de donnée PhPMyAdmin via du PHP

Projet fait en Groupe

Voici un projet PhP effectué en tant qu'Ap pendant mon BTS SIO

Ce projet a permis de faire du php avec une base donnée PhPMyAdmin et comment manipuler une base de donnée via du PHP

Dans ce projet on a acquis plusieurs compétences notamment Bootstrap un framework CSS qui à été la première phase du projet. C'est un projet séparer en deux phase, la première phase était la contruction du site et la création de plusieur page commune pour différentes catégorie comme par exemple pour les personnages ou bien ou les catégories de chaque personnage.

Ensuite la deuxième phase consiste à implémenter le php et une base de donnée pour rendre le site dynamique. Le php m'a permis notamment d'ajouter certaine fonctionnalité comme ajouter/modifier/supprimer des personnages via un pannel administrateur mais aussi optimisé le projet en centralisant la barre de navigation, le footer et la sidebar.
